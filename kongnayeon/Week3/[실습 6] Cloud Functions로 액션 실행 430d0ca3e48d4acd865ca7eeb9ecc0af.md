# [실습 6] Cloud Functions로 액션 실행

### **Cloud Functions란?**

별도의 서버 없이 이벤트를 처리할 수 있습니다. 액션(Action)에 등록한 코드는 트리거(Trigger)와 연결되어 클라우드 환경에서 발생하는 모든 이벤트에 응답합니다. 사용한 컴퓨팅 시간만큼만 비용이 발생하여 효율적입니다.
