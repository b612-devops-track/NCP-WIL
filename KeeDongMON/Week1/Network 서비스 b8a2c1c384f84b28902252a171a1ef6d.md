# Network 서비스

## Load Balancer

- 서버 앞단에서 트래픽 분산
- 서버2개 도메인에 바인딩 되어 제공

## DNS

## CDN+ / GCDN

컨텐츠를 Caching하여 보다 빠르고 안정적으로 전송

ex : 부산에서 서울에 있는 데이터를 요청할 때 처음에는 서울로 요청하고 이후에는 부산에 보관하여 딜레이를 줄

## IPSEC VPN

사내 vpn장비와 NCP vpn장비를 다이렉트로 연결

## NAT Gateway

- 비공인 IP를 가진 다수의 서버에게 대표 공인 IP를 이용한 외부 접속을 제공

## Global Route Manger

Load Balancer과 달리 여러 지역으로 트래픽 분산 가능
